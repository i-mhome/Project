# 전지환 프로젝트 제안서

## 1. 프로젝트 이름

로보 자비스(회계 정산 ver)

---

## 2. 한 문장 설명

동아리 행사 참여 명단과 체크 명단을 직접 입력하거나 이미지 OCR로 읽어 미입금자를 찾고, 1인당 정산 금액과 정산 결과 CSV 파일을 자동으로 만들어주는 프로그램입니다.

---

## 3. 풀려는 문제 또는 만들려는 이유

저는 현재 로보라는 과동아리 회장으로 활동하고 있습니다. 동아리에서는 행사나 회식이 있을 때마다 참여자 명단을 확인하고, 전체 금액을 인원 수로 나눈 뒤, 누가 입금했는지 다시 비교해야 합니다. 기존에는 엑셀이나 카카오톡 체크 명단을 보면서 직접 비교했기 때문에 인원이 많아질수록 미입금자 확인이 번거로웠습니다. 그래서 참여 명단과 체크 명단을 입력하거나 캡쳐 이미지에서 읽어와 자동으로 미입금자를 찾고, 1인당 금액과 결과표를 만들어주는 정산 보조 프로그램을 만들고자 합니다.

---

## 4. 핵심 기능 및 우선순위

### [MUST] 반드시 구현할 기능

1. **직접 입력 방식 정산 기능**
   - 전체 명단과 체크된 사람 명단을 직접 입력받아 리스트로 변환합니다.
   - 전체 명단에는 있지만 체크 명단에는 없는 사람을 미입금자로 분류합니다.

2. **1인당 정산 금액 계산 기능**
   - 행사 총 금액을 입력받고 전체 인원 수로 나누어 1인당 납부 금액을 계산합니다.
   - 숫자가 아닌 값이 입력되면 `try-except`로 오류를 처리하고 다시 입력받습니다.

3. **정산 결과 출력 기능**
   - 전체 인원, 체크된 사람, 미입금자, 전체 인원 수, 1인당 금액을 터미널에 출력합니다.

4. **CSV 저장 기능**
   - 정산 결과를 `settlement_result.csv` 파일로 저장합니다.
   - 이름, 상태, 납부금액을 표 형태로 저장하여 엑셀에서 확인할 수 있게 합니다.

5. **OCR 이미지 정산 기능**
   - 참여 명단 이미지 폴더와 체크 명단 이미지 폴더를 입력받습니다.
   - 각 폴더 안의 이미지 파일을 EasyOCR로 읽어 이름 리스트로 변환합니다.
   - OCR로 읽은 명단을 기존 정산 처리 함수에 연결하여 미입금자를 계산합니다.

### [NICE] 있으면 좋은 기능

1. **전체 명단에 없는 체크 이름 경고 기능**
   - 체크 명단에는 있지만 전체 명단에는 없는 이름을 따로 출력하여 오타나 잘못된 입력을 확인할 수 있게 합니다.

2. **여러 이미지 자동 처리 기능**
   - 이미지 파일명을 하나씩 입력하지 않고, 폴더 안의 `.png`, `.jpg`, `.jpeg` 파일을 자동으로 읽습니다.

3. **중복 이름 제거 기능**
   - 여러 이미지에서 같은 이름이 반복해서 읽히는 경우 한 번만 저장합니다.

### [LATER] 시간이 남으면 추가할 기능

1. **동명이인 구분 기능**
   - 이름만으로 구분이 어려운 경우 학번 일부나 고유 번호를 함께 사용하도록 확장합니다.

2. **OCR 결과 수정 기능**
   - OCR 인식 결과가 틀렸을 때 사용자가 직접 수정할 수 있도록 합니다.

3. **회비 입출금 및 잔액 관리 기능**
   - 행사 정산뿐만 아니라 동아리 회비 입금, 출금, 현재 잔액까지 관리할 수 있도록 확장합니다.

# README

## 1. 프로젝트 소개

이 프로젝트는 동아리 행사 정산을 돕기 위한 Python 프로그램입니다.

동아리 행사 후 전체 참여 명단과 입금 또는 체크 완료 명단을 비교하여 미입금자를 자동으로 찾아줍니다. 또한 행사 총 금액을 입력하면 1인당 납부 금액을 계산하고, 정산 결과를 CSV 파일로 저장할 수 있습니다.

기본적으로 직접 입력 방식으로 정산할 수 있으며, 추가로 EasyOCR을 활용하여 이미지 폴더에 있는 명단 캡쳐본을 읽고 정산에 활용할 수 있도록 구현했습니다.

---

## 2. 프로젝트를 만들게 된 이유

저는 동아리 활동을 하면서 행사나 회식 후 정산을 해야 하는 상황이 자주 있었습니다.

기존에는 참여자 명단과 입금자 명단을 직접 비교해야 했고, 인원이 많아질수록 누가 입금했는지 확인하는 과정이 번거로웠습니다. 또한 사람이 직접 확인하다 보면 누락이나 오타가 생길 가능성도 있었습니다.

그래서 전체 명단과 체크 명단을 비교하여 미입금자를 자동으로 찾고, 정산 결과를 파일로 저장할 수 있는 프로그램을 만들고자 했습니다.

---

## 3. 주요 기능

### 3.1 직접 입력 정산 기능

사용자가 전체 명단과 체크된 사람 명단을 직접 입력하면, 프로그램이 두 명단을 비교하여 미입금자를 찾아줍니다.

예시 입력 형식은 다음과 같습니다.

~~~text
김철수, 이영희, 박민수
~~~

입력된 문자열은 쉼표를 기준으로 나누어 Python 리스트로 변환됩니다.

---

### 3.2 OCR 이미지 정산 기능

EasyOCR을 사용하여 이미지 속 글자를 읽고, 이를 명단으로 변환합니다.

사용자는 참여 명단 이미지가 들어 있는 폴더와 체크 명단 이미지가 들어 있는 폴더를 입력합니다. 프로그램은 각 폴더 안의 `.png`, `.jpg`, `.jpeg` 이미지 파일을 자동으로 찾고, OCR을 통해 이름을 읽어옵니다.

---

### 3.3 미입금자 계산 기능

전체 명단에는 있지만 체크 명단에는 없는 사람을 미입금자로 판단합니다.

~~~text
전체 명단 - 체크 명단 = 미입금자
~~~

이를 통해 사용자는 누가 아직 입금하지 않았는지 빠르게 확인할 수 있습니다.

---

### 3.4 1인당 금액 계산 기능

행사 총 금액을 입력하면 전체 인원 수로 나누어 1인당 납부 금액을 계산합니다.

숫자가 아닌 값을 입력했을 경우에는 `try-except`를 사용하여 오류를 처리하고, 다시 입력하도록 했습니다.

---

### 3.5 전체 명단에 없는 이름 확인 기능

체크 명단에는 있지만 전체 명단에는 없는 이름을 따로 출력합니다.

이 기능은 이름 오타, OCR 인식 오류, 잘못된 입력을 확인하는 데 도움이 됩니다.

---

### 3.6 CSV 저장 기능

정산 결과를 `settlement_result.csv` 파일로 저장합니다.

CSV 파일에는 다음과 같은 정보가 저장됩니다.

~~~text
이름, 상태, 납부금액
~~~

CSV 파일은 엑셀에서 열 수 있기 때문에 정산 결과를 기록으로 보관하거나 다시 확인하기 쉽습니다.

~~~

처음에는 하나의 함수에 많은 기능을 넣으려고 했지만, 코드가 길어지고 복잡해졌습니다. ChatGPT의 도움을 받아 입력, OCR 처리, 정산 계산, 결과 출력, CSV 저장 기능을 함수 단위로 분리했습니다.

### 8.4 OCR 라이브러리 선택

처음에는 Tesseract OCR도 고려했지만, 한글 이름 인식이 불안정하고 설정 과정이 복잡했습니다. ChatGPT를 통해 EasyOCR을 알게 되었고, Python에서 사용하기 쉽고 한글 인식을 지원한다는 점에서 EasyOCR을 선택했습니다.

### 8.5 오류 해결 및 코드 이해

파일 경로 오류, 숫자 입력 오류, OCR 인식 문제를 해결하는 과정에서 ChatGPT의 도움을 받았습니다.

다만 AI가 작성한 코드를 그대로 복사해서 사용하지 않고, 직접 한 줄씩 입력하고 실행하면서 이해하려고 했습니다. 이해가 되지 않는 부분은 다시 질문하고, 직접 주석을 달아가며 코드의 동작을 정리했습니다.

## 9. 실행 결과

## 10. 한계점

현재 프로그램은 이름 문자열을 기준으로 명단을 비교합니다. 따라서 동명이인이 있을 경우 서로 다른 사람을 구분하기 어렵습니다.

또한 OCR 기능은 이미지 품질, 글자 크기, 파일명, 경로 형식에 영향을 받을 수 있습니다. OCR이 이름을 잘못 인식하는 경우 현재 버전에서는 사용자가 직접 결과를 수정하는 기능이 부족합니다.

## 11. 향후 개선 방향

향후에는 다음과 같은 기능을 추가하고 싶습니다.

1. OCR 결과 수정 기능
   - OCR로 읽은 이름을 화면에 보여주고, 사용자가 잘못된 이름을 수정한 뒤 정산할 수 있도록 개선

2. 동명이인 구분 기능
   - 이름뿐만 아니라 학번 일부나 회원 번호를 함께 사용하여 사람을 구분

3. GUI 버전 확장
   - 터미널 입력 방식이 아니라 버튼과 파일 선택창이 있는 화면으로 개선

4. 회비 입출금 및 잔액 관리 기능
   - 단순 행사 정산뿐만 아니라 동아리 전체 회비 관리 기능으로 확장
