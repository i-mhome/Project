# 전지환 프로젝트 제안서

---

## 1. 프로젝트 이름

로보 자비스(회계 ver)

---

## 2. 한 문장 설명

동아리 회비 입출금 내역을 기록해 현재 잔액을 자동으로 계산하고, 행사 참석자와 입금자를 비교해 정산 금액과 미입금자 명단을 정리해주는 프로그램입니다.

---

## 3. 풀려는 문제 또는 만들려는 이유

저는 현재 로보 라는 과동아리 회장으로 활동하고 있습니다. 
동아리는 회비로 운영을 하고, 행사를 할때 항상 회비로 결제를 합니다. 그러나 행사가 자주 있고 그 전체 계산은 보통 제가 하는편입니다. 이때 행사나 회식 이후에는 참석자를 확인하고, 1인당 정산 금액을 계산한 뒤, 누가 입금했는지 다시 확인을 해야합니다. 기존에는 엑셀표를 공유하고 직접 명단을 비교하면서 정산을 진행하였는데, 인원이 많아질수록 참석자 확인, 입금 확인, 미입금자 정리가 번거로웠습니다. 그리고 행사를 진행할때 남은 회비 잔액을 확인하고 행사를 정하기 때문에 회비 잔액도 알 필요가 있습니다. 고로 회비 입출금 내역과 행사 정산 정보를 입력하면 현재 잔액, 1인당 정산 금액, 미입금자 명단을 자동으로 만들려고 합니다.

---
## 4. 핵심 기능 3가지

### 1) 회비 입출금 내역 기록 기능

동아리 회비의 입금과 출금 내역을 입력하고 저장합니다.

### 2) 현재 잔액 자동 계산 기능

저장된 입금 출금 내역을 바탕으로 현재 동아리 회비 잔액을 자동으로 계산하도록 합니다.

### 3) 행사, 회식 정산 및 미입금자 확인 기능

행사 총금액과 참석자 정보를 바탕으로 1인당 정산 금액을 계산하고, 입금 완료자와 미입금자를 구분합니다.

---
## 5.예상 사용자

동아리 회장, 총무, 학회장, 소모임 운영자 등 여러 명의 회비와 행사 정산을 관리해야 하는 사람입니다. 저 또한 포함입니다.
---
## 6.결과물 형태

### CLI 프로그램

이 프로젝트는 화면 디자인보다 회비 잔액 계산, 정산 금액 계산, 미입금자 분류 기능이 중요하므로 CLI 형태가 적절하다고 판단하였습니다. 터미널에서 메뉴를 선택하고, 필요한 정보를 입력하면 결과가 출력되는 방식으로 구현할려고 합니다. 이때 VS code의 Jupyter notebook 환경에서 실행할 수 있는 터미널형 프로그램으로 구현하려고 합니다.

## 8. 기대효과

이 프로그램을 이용하면 따로 내가 엑셀로 회계 정리를 하고 계좌를 보면서 잔액 출처를 정리하고 할 필요가 없어지고, 행사 정산 시 참석자와 입금자를 직접 비교하는 시간을 줄일 수 있습니다. 또한 미입금자를 확인하는일이 항상 귀찮은데 이걸 자동으로 정리할 수 있어 회장이나 총무의 반복적인 정산 업무를 줄이는 데 도움이 됩니다.
